# Features

Discover the powerful tools and systems that make Lilia a leading roleplay framework.

---

<h3 style="margin-bottom: 5px; font-weight: 700;">Overview</h3>
<div style="margin-left: 20px; margin-bottom: 20px;">
  <p>Lilia is built with a focus on developer flexibility and player performance. It provides a robust suite of built-in systems that handle everything from character persistence to advanced UI interactions, allowing you to focus on creating unique roleplay experiences.</p>
</div>

---

<h3 style="margin-bottom: 5px; font-weight: 700;">Core Systems</h3>
<div style="margin-left: 20px; margin-bottom: 20px;">
  <details class="realm-shared no-icon">
    <summary>Advanced Character Management</summary>
    <div class="details-content">
      <p>Multi-character support with shared or isolated data. Includes specialized systems for factions, classes, attributes, and model-specific configurations.</p>
    </div>
  </details>

  <details class="realm-shared no-icon">
    <summary>Grid Inventory & Items</summary>
    <div class="details-content">
      <p>A high-performance grid inventory system supporting item weight, drag-and-drop, containers (bags), and extensive item metadata for dynamic behavior.</p>
    </div>
  </details>

  <details class="realm-shared no-icon">
    <summary>Modular Architecture</summary>
    <div class="details-content">
      <p>Clean separation of concerns with a powerful module system. Easily enable, disable, or extend functionality without touching the core framework code.</p>
    </div>
  </details>
</div>

<h3 style="margin-bottom: 5px; font-weight: 700;">Developer Experience</h3>
<div style="margin-left: 20px; margin-bottom: 20px;">
  <details class="realm-shared no-icon">
    <summary>Automated Generators</summary>
    <div class="details-content">
      <p>Built-in web-based generators for quickly creating items, factions, and classes with localized name and description support.</p>
    </div>
  </details>

  <details class="realm-shared no-icon">
    <summary>Comprehensive Hooking</summary>
    <div class="details-content">
      <p>Hundreds of documented hooks allow developers to intercept and modify framework behavior at every critical point.</p>
    </div>
  </details>
</div>

<h3 style="margin-bottom: 5px; font-weight: 700;">User Interface</h3>
<div style="margin-left: 20px; margin-bottom: 20px;">
  <details class="realm-shared no-icon">
    <summary>Sleek Modern Derma</summary>
    <div class="details-content">
      <p>Custom-themed Derma controls providing a consistent, sleek aesthetic across all menus, inventories, and admin tools.</p>
    </div>
  </details>

  <details class="realm-shared no-icon">
    <summary>Responsive Chatbox</summary>
    <div class="details-content">
      <p>Integrated chat system with support for custom channels, command autocomplete, and rich text formatting.</p>
    </div>
  </details>
</div>
